# OMO Ecological Informatics Project

## Vision

To develop a reproducible ecological informatics framework that transforms traditional vegetation inventory data into machine learning-ready ecological datasets for biodiversity analysis, habitat classification, explainable artificial intelligence, and conservation decision support.

---

## Long-Term Goal

Build one of the most comprehensive open ecological informatics repositories for tropical riparian forests in West Africa.

---

## Current Dataset

Omo Biosphere Reserve

Nigeria

Datasets available

- Tree species abundance
- Shrub abundance
- Ground flora abundance
- Tree structural measurements
- Additional datasets to be incorporated later
